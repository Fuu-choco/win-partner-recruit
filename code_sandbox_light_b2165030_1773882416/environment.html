Work Style

当社の勤務制度は3種類。出退勤が完全自由な「フルフレックス制度」、コアタイム内で柔軟な働き方が可能な「フレックスタイム制度」、残業を一切認めず定時勤務を徹底する「ワークライフバランス制度」を用意。いずれも個人のライフスタイルやキャリアステージに合わせて選択できます。

時短・海外リモート・産休育休・業務委託など、多様な働き方に対応しています。一人ひとりの事情や希望に合わせた柔軟な勤務形態を整えています。

肩書きや部署に縛られない。それが、ウィンパートナーの働き方です。フリーアドレスによって部署間・階層間の交流が自然と生まれています。

出勤・リモート・MIX、自分に合ったスタイルで働けます。毎日オフィスに来たい人も、自宅で集中したい人も、その日によって使い分けたい人も。働く場所は、自分で決めてください。

服装の規定はありません。クライアント訪問など場面に応じた対応はしつつ、社内やリモートワーク時は完全に自由です。自分らしいスタイルで働いてください。

昼食後に15〜30分の休憩を推奨しています。短時間の休息が午後の集中力を高めることは科学的にも証明されています。休む時間も、仕事のうちだと考えています。
