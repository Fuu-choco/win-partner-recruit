Model Case

コードも営業も、
未経験の自分が管理職になるまで。

プログラミングも営業も、入社前は何もできませんでした。でも研修を通じてコードを書く力だけでなく、クライアントと向き合うコミュニケーション力や提案力まで自然と身についていきました。「技術だけじゃない人間」に育ててもらえたことが、今の自分の一番の強みです。

エンジニア → チームリーダー

Haruto Sasaki

  • 1年目 研修でエンジニアの基礎を習得
  • 2年目 実案件に参加・営業同行も経験
  • 3年目 クライアントへの提案を単独で担当
  • 4年目 後輩エンジニアのメンターに就任
  • 5年目 チームリーダーとして管理職へ昇格

毎朝憂鬱だった通勤が、
今は楽しみに変わりました。

前職では毎朝、会社に行くのが憂鬱でした。決まった時間、決まった場所、決まった仕事。「これが一生続くのかな」と思っていた頃に転職を決意。ウィンパートナーでは働く時間も場所も自分で選べて、最初は戸惑うくらいでした。今は仕事が楽しくて、休日もプライベートを全力で楽しめています。

一般事務 → プロジェクトマネージャー

Sara Inoue

  • 1年目 フレックス制度を活用しながら業務習得
  • 2年目 チームの進行管理を任されるように
  • 3年目 リモートワークを活用しながら成果を出す
  • 4年目 複数プロジェクトを同時に管理
  • 5年目 プロジェクトマネージャーとして管理職へ

「未経験はお断り」の壁を
ウィンパートナーが壊してくれた。

エンジニアになりたくて独学で勉強したものの、「未経験者は採用されづらい」「伏線を取りづらい」という壁に何度もぶつかりました。ウィンパートナーの研修制度と出会い、9ヶ月でゼロから実務レベルまで到達。夢を諦めなくてよかったと、今は心から思えます。いい環境があれば、道は必ず開けます。

飲食業 → エンジニア → エンジニアリング責任者

Yuki Tawara

  • 1年目 9ヶ月研修でプログラミングを習得
  • 2年目 実案件の開発に参加
  • 3年目 新機能の設計・実装をリード
  • 4年目 チームの技術方針を策定
  • 5年目 開発リーダーとして管理職へ昇格
